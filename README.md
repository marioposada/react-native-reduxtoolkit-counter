# Description
App developed to implement redux.

## Build Setup

``` bash
# clone project
$ git clone git@github.com:marioposada/react-native-redux-implement.git

# install dependencies
$ npm install

# serve with host at localhost:19000
$ npm start
$ yarn start
```
